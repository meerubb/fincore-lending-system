# FinCore — Professional Lending System for Banks
Replacing manual ledgers with a secure, scalable relational lending ecosystem.

## Overview
FinCore is a full-featured Loan Management System built using SQL and 
relational database principles. Designed for the banking and lending sector, 
it automates borrower onboarding, repayment tracking, collateral management, 
risk classification, and fraud auditing through a normalized 3NF schema with 
ACID-compliant integrity controls.

## Core Features
- **Borrower Management** — Full CRUD lifecycle for borrower profiles and credit histories
- **Loan & Asset Tracking** — Links collateral (real estate, vehicles) to specific loan accounts
- **Transaction Engine** — Automated installment logging with overdue penalty calculations
- **Risk Classification** — Algorithmic Low/Medium/High risk categorization by credit score and income
- **Integrity Auditing** — Immutable trigger-based audit logs on sensitive tables for fraud prevention
- **Analytics Dashboard** — Backend BI views for interest revenue summaries and overdue account identification

## Tech Stack
`SQL` · `Relational Schema Design` · `3NF Normalization` · `Triggers` · `Views` · `ACID Compliance`
